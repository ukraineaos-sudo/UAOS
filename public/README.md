# Публічна папка ресурсів / Public Assets Folder

## 🏆 Фото нагороди "Лідер Року" / Award Image
Будь ласка, завантажте ваше фото нагороди безпосередньо в цю папку (`/public/`) через файловий менеджер у редакторі коду AI Studio.
* Назва файлу має бути: **`leader_award.jpg`**
* Щойно файл з'явиться в папці, він миттєво відобразиться на головному банері!

---

## 🎨 Логотип сайту / Website Logo
Будь ласка, завантажте ваш офіційний логотип безпосередньо в цю папку (`/public/`) через файловий менеджер у редакторі коду.
* Підтримувані формати та відповідні назви файлів:
  * **`logo.png`** (рекомендовано з прозорим фоном)
  * **`logo.svg`**
  * **`logo.jpg`**
  * **`logo.jpeg`**
* Логотип автоматично з'явиться у верхній панелі (Header) та у підвалі (Footer) сайту замість стандартної іконки щита!
